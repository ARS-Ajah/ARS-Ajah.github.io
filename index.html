Discord Nitro Generator

This webpage is a webpage to generate nitro links

FAQ:

How does it works?

These script will make an 18 CHARACTER alphanumeric string. This string will then send a GET request to

https://discordapp.com/api/v9/entitlements/gift-codes/{Code}?with_application=false&with_subscription_plan=true

the {Code} is the 18 Charater string. If the GET Request is returned with a Status code of 200 OK.

Then it will send the url to your server with a https://discord.gift/{Code}

Is this website stealing my webhook token?

Nope!, We only store webhook URL on the client side

What happened on the backend?

The backend is the place for us to send an GET Request and sending with the webhook
