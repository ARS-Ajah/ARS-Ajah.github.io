Infinite String Generator

This script will infinitely generate strings of text, each with 18 randomly generated alphanumeric values.

These strings will then be sent to a function that will send a GET request to the URL "https://discordapp.com/api/v9/entitlements/gift-codes/{Code}?with_application=false&with_subscription_plan=true" where the {Code} is the randomly generated string.

If the GET request returns with a code of 200 OK, then you will get your code

REMEMBER: We don't consider this to be working in fact the chance of this happening is 1/5000000000000000000 or 1/5 octillion so we are not recommending this method
